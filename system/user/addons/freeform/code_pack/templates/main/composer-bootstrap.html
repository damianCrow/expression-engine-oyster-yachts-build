Composer-based Form with Bootstrap Template  Freeform Pro only

This page contains an example form built with Freeform Pro using the Composer-based approach, formatted and styled with Bootstrap in a custom Composer Template. Composer Templates can be created inside the Freeform control panel and assigned to Composer-based form(s), or coded directly into an EE template. This example has the Composer Template code directly inside this EE Template.
